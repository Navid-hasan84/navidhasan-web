## Navidhasan.com Phase 1

This project is the Phase 1 static frontend for `navidhasan.com`.

The goal is to build a moon-themed public website that can run on Cloudflare Pages without backend cost.

### Current architecture

- Frontend: Next.js App Router
- Hosting target: Cloudflare Pages
- Output mode: Static export
- Backend: Not included in Phase 1
- Database: Not included in Phase 1
- API dependency: None for public pages

### Phase 1 priorities

- Static public pages
- Responsive desktop and mobile navigation
- Moon-inspired visual identity
- SEO-friendly page structure
- Fast Cloudflare deployment
- No backend dependency
- No database dependency
